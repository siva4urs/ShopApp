# ShopApp Monorepo

This repository contains a monorepo scaffold for ShopApp: a React TypeScript web app, an Expo React Native mobile app, a GraphQL server, shared types, Vitest for unit testing and Playwright for E2E testing.

Getting started:

1. Install pnpm: `npm i -g pnpm`
2. Install dependencies: `pnpm install`
3. Start GraphQL server: `pnpm dev:graphql`
4. Start web: `pnpm dev:web`
5. Start mobile (expo): `pnpm dev:mobile`

Testing:
- Unit tests: `pnpm test`
- E2E (Playwright): `pnpm e2e`
